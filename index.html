Insta485

Insta485 is an Instagram-like social media platform with both client-side and server-side dynamic pages that enables authenticated users to create posts, view posts, like posts, comment on posts, follow other users, and see a list of all users they are not following.

ask485

ask485 is a search engine that can return ranked results from 2935 sources in 0.4 seconds. It leverages the MapReduce framework to precompute information about the sources and uses the PageRank algorithm (with a user-customizable weight parameter) to return relevant results for each query.

build

Made with HTML, Jinja, CSS, Python, and Flask
